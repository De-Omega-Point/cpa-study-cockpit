# CPA Study Cockpit

Sub-brand: De-Omega-Point Project

A GitHub Pages-ready React + TypeScript + Vite CPA mastery platform for Illinois CPA candidates.

## Included
- Landing-page onboarding, personalized dashboard, profile, XP, analytics, theme toggle and mobile nav
- Illinois CPA pathway tracker
- CPA Academy lesson engine with 40 generated lessons
- TEST MASTER™ with 200 CPA-style questions
- Vocabulary Master with 500 terms
- Case Study Library with 10 major cases
- War Room™ readiness and mock exam command centre

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to GitHub Pages
Update `base` in `vite.config.ts` to match your repo name, then:
```bash
npm run deploy
```
